# MEMIT 사용 설명서 (User Manual)

MEMIT에 오신 것을 환영합니다! 이 가이드는 AI를 활용하여 숫자를 쉽고 재미있게 외우는 방법을 안내합니다.

## 1. 숫자 암기하기
1.  **숫자 입력**: 메인 화면의 입력창에 기억하고 싶은 숫자(도어락 비번, 카드 번호 등)를 입력합니다.
2.  **변환하기**: **[Brain]** 버튼(또는 Enter)을 누르면 AI가 숫자를 단어로 변환하고 애니메이션과 함께 스토리를 생성합니다.

## 2. 내 입맛에 맞는 단어로 바꾸기
AI가 추천해준 단어가 마음에 들지 않는다면 직접 바꿀 수 있습니다.
*   **단어 클릭**: 결과 창의 단어 카드(예: '총알')를 클릭하세요.
*   **교체**: 나타나는 후보 단어 리스트 중 가장 기억에 잘 남는 단어를 선택하세요.
*   **스토리 재생성**: 단어를 바꾸면 나타나는 **[🔄 단어 맞춤 스토리 재생성]** 버튼을 눌러, 바뀐 단어에 맞게 이야기를 다시 만듭니다.

## 3. 특정 단어 고정하기 (Pin/Lock) 🔒
중요하거나 이미 외운 단어는 바뀌지 않도록 고정할 수 있습니다.
*   **자물쇠 클릭**: 단어 카드 오른쪽의 자물쇠 아이콘을 클릭하세요.
*   **고정 효과**: 고정된 단어는 상단의 자물쇠가 잠기며, 다시 '변환하기'를 눌러도 다른 단어로 바뀌지 않습니다.
*   **팁**: 수동으로 단어를 교체하면 해당 단어는 자동으로 고정됩니다.

## 4. AI 이미지로 시각화하기 🎨
스토리를 다 완성했다면 이를 그림으로 확인하세요.
*   **타입 선택**: '1컷 요약' 또는 '4컷 스토리' 중 원하는 형식을 고릅니다.
*   **이미지 생성**: 하단의 **[🎨 그림으로 보기]** 버튼을 누르면 DALL-E 3가 스토리를 멋진 이미지로 그려줍니다.
